#webchat-shopping
微信小程序 商城

##前言
小程序入门，仿照一个商城的小程序练手

## 功能
- [x] 首页
- [x] 搜索
- [x] 分类
- [x] 购物车
- [x] 个人中心
- [x] 商品列表
- [x] 商品详情
- [x] 订单
- [x] 地址管理
